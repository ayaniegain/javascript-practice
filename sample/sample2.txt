### 1. **Array Creation and Manipulation**

- Create an array with numbers 1 to 5.  
- Add an item to the end of an array and remove the last item.
- Add an item to the beginning of an array and remove the first item.
- Merge two arrays `[1, 2]` and `[3, 4]` into one.
- Extract a portion of an array without modifying the original.
- Remove elements from an array, add elements, or replace them in the array.

### 2. **Searching and Indexing**

- Find the first even number in `[3, 5, 6, 7]`.
- Find the index of the number `7` in `[2, 5, 7, 8]`.
- Get all numbers greater than 10 from an array.

### 3. **Transforming Arrays**

- Create a new array by doubling each number in `[1, 2, 3]`.
- Calculate the sum of `[5, 10, 15]`.
- Sort `[2, 5, 1, 9, 7]` in ascending order.
- Reverse an array.

### 4. **Checking Elements**

- Check if `8` exists in `[3, 5, 8, 10]`.
- Check if all numbers in `[5, 8, 10]` are even.
- Check if any number in `[4, 7, 9]` is greater than 8.

### 5. **Complex Exercises**

- Flatten a nested array `[1, [2, [3, 4]], 5]`.
- Remove duplicates from an array `[1, 2, 2, 3, 4, 4, 5]`.
- Find the common elements between two arrays `[1, 2, 3]` and `[2, 3, 4]`.
- Convert an array `['a', 'b', 'c']` to an object `{a: 1, b: 1, c: 1}`.
- Convert an object `{a: 1, b: 2}` to an array of key-value pairs `[["a", 1], ["b", 2]]`.

### 6. **Algorithmic Challenges**

- Find the maximum sum of a subarray in `[4, -1, 2, 1, -5, 4]`.
- Rotate the array `[1, 2, 3, 4, 5]` by `k` positions.
- Generate the first `n` numbers of the Fibonacci sequence and store them in an array.
- Find all pairs of numbers in `[2, 4, 3, 6, 8]` that sum to `10`.
- Find the longest increasing subsequence in `[10, 22, 9, 33, 21, 50, 41, 60, 80]`.

### 7. **Practice Exercises**

- Write a function to find the largest element in an array.
- Write a function to reverse an array without using `.reverse()`.
- Count the frequency of each element in an array.
- Remove all falsy values from `[0, 1, false, 2, '', 3]`.
- Divide an array `[1, 2, 3, 4, 5]` into smaller arrays of length `2`.
